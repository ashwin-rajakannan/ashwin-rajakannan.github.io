# Test Guide

## Running Backend Unit Tests

1. **Install dependencies:**
   - `pip install -r requirements.txt`

2. **Run tests:**
   - `python -m unittest discover tests`

## Test File
- `tests/test_sentiment_model.py`: Contains unit tests for sentiment analysis logic.
